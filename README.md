# PowerModeTray

윈도우11에서 사라진 작업표시줄에서 전원 모드를 간단히 구현.  
관련 내용 및 아이디어는 [BatteryIcon](https://github.com/psjoel02/BatteryIcon) 프로젝트를 참고하였습니다.
